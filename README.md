# Full Stack Web Application (Vue.js & Node.js)

## Overview
This project is a simple full-stack web application built using Vue.js for the frontend and Node.js for the backend.  
It was created as a learning project to demonstrate basic full-stack development skills, including frontend UI development, backend API design, and data handling.

## Features
- Simple and responsive frontend built with Vue.js  
- Backend RESTful APIs implemented using Node.js  
- Basic CRUD functionality (Create, Read, Update, Delete)  
- Clear separation between frontend and backend logic  

## Technologies Used
- **Frontend:** Vue.js, HTML, CSS  
- **Backend:** Node.js  
- **Tools:** Git, VS Code

## Purpose
The purpose of this project is to practice building a complete web application from scratch and to gain hands-on experience with the full development lifecycle.
